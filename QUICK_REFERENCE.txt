# 🚀 QUICK REFERENCE - Google Sign-In Error 10

## ❌ Problem
```
Google sign in failed: 10
```

## ✅ Solution
Add your app's **SHA-1 fingerprint** to Firebase Console

---

## 📝 3-Step Fix

### 1️⃣ Get Fingerprint (Terminal)
```bash
cd /Users/deepakanish/AndroidStudioProjects/cloudvault
./gradlew signingReport
```
**Look for:** `SHA1: XX:XX:XX:...` and copy it

### 2️⃣ Add to Firebase
```
https://console.firebase.google.com/
  → self-drive-17ee0 (project)
  → ⚙️ Project Settings
  → Your apps
  → com.cloudvault.cloudvault
  → Add fingerprint (paste SHA1)
  → Save
```

### 3️⃣ Rebuild App
```bash
./gradlew clean build
adb uninstall com.cloudvault.cloudvault
./gradlew installDebug
```

**⏳ Wait 10-15 minutes after step 2 before rebuilding in step 3**
